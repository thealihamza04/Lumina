# Design System & Aesthetics

This project prioritizes **Minimalist Visual Excellence**. The UI should feel clean, functional, and high-fidelity without unnecessary visual clutter.

## Guidelines for Agents
1. **Never use generic colors**: Use curated, harmonious palettes. Stick to professional, muted tones for controls to keep focus on the gradient output.
2. **Minimalism**: Avoid `backdrop-blur`, heavy shadows, and "extra" decorative elements in UI controls. Focus on clean lines and clear purpose.
3. **Minimal Animations**: Use only essential micro-animations (e.g., subtle hover shifts). Avoid complex or distracting animations that don't add functional value.
4. **Typography**: Stick to modern, clean sans-serif fonts. Use clear hierarchies with appropriate weight and spacing.
5. **Component Polish**: Focus on high-quality interaction states (hover, focus, active) using solid colors and clear borders rather than shadows or blurs.

## Key UI Components
- **Control Panel**: A high-fidelity sidebar for all settings.
- **Layers**: Cards with blend mode indicators and visibility toggles.
- **Color Stops**: Precision-engineered sliders with real-time color feedback.

## Premium Checklist
- [ ] Does it maintain a clean, minimalist aesthetic?
- [ ] Are animations minimal and non-distracting?
- [ ] Did you avoid unnecessary blurs and shadows?
- [ ] Is the spacing consistent and elegant?
