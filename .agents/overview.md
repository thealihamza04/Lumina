# Project Overview: Gradient Generator

A premium, high-fidelity web application for creating beautiful CSS gradients visually. Supports linear, radial, and conic gradients with multi-layer stacking capabilities.

## Core Purpose
The goal of this project is to provide a professional-grade tool for designers and developers to experiment with complex gradients, preview them in real-time, and export the resulting CSS.

## Key Features
- **Multi-Layer System**: Stack multiple gradient layers to create depth and complex visual effects.
- **Visual Editor**: Interactive color stop placement and adjustment.
- **Gradient Types**: Full support for Linear, Radial, and Conic gradients.
- **Blending & Effects**: Per-layer blur and blend mode controls.
- **Minimalist UI**: Modern, clean design focused on functional clarity, with minimal animations and a distraction-free interface.
