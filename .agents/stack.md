# Technology Stack

The project is built with modern, industry-standard web technologies focused on performance and developer experience.

## Core Framework
- **Next.js (App Router)**: React framework with server-side rendering and routing.
- **TypeScript**: Static typing for enhanced maintainability and error checking.

## UI & Styling
- **Tailwind CSS (v4)**: Utility-first CSS framework for rapid and consistent styling.
- **Radix UI**: Primitive components for accessible, high-quality UI foundations (Selects, Sliders, Switches, Tabs).
- **Lucide React**: Modern, consistent icon set.
- **Minimalist Design**: Focused on clean borders and functional clarity.

## Utilities
- **Colord**: Lightweight and high-performance color manipulation library.
- **React-Colorful**: Minimalist and accessible color picker component.
- **Clsx & Tailwind-merge**: Utilities for dynamic and conflict-free class management.
- **Recharts**: For any data visualization needs (if applicable).
- **Zod & React Hook Form**: Robust form validation and state management.
